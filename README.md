# Demo project for React SSR

Until now, I have ignored the Server-Side-Rendering / Server-Side-Routing (SSR) paradigm
because React 18 docs seemed to indicate I should steer clear unless using Next.js. But
the API is there, and with the release of React 19 I thought it was high time I addressed
my anxiety around all things SSR.

I also wanted to see if I could make use of Server Components, another equally mysterious
_here be dragons_ React API that comes with the same warnings around using it via Next.js.
I hate being told what to do at the best of times, so let's see what I can come up with...

## Run me

Project is scaffolded using the magnificent [Vite](https://vite.dev).

```
npm run dev
```
